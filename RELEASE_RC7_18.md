# BBLOTTO PRO V2 STABLE RC7-18

문자간다 엑셀 업로드에서 셀 내부 줄바꿈이 삭제되는 환경에 맞춘 최종 호환 패치입니다.

## 핵심 변경
- [*1*]~[*4*] 셀 값은 모두 한 줄로 저장
- 추천번호는 [*3*] 1~5조합, [*4*] 6~10조합 + 마무리 문구로 분리
- 조합 사이를 `/` 로 구분해 줄바꿈이 없어도 번호가 붙지 않게 처리

## 문자간다 입력창 템플릿
아래처럼 문자간다 내용 입력창에 직접 넣으세요. 줄바꿈은 이 입력창에서 유지됩니다.

```
[*1*]

[*2*]

[*3*]
[*4*]
```

※ 엑셀 셀 안에 줄바꿈을 넣는 방식은 문자간다 업로드 과정에서 삭제될 수 있어 사용하지 않습니다.
