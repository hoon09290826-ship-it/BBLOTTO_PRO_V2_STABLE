# BBLOTTO PRO V2 STABLE RC7-16

## 목적
문자간다 원본 샘플 XLS 기준으로 엑셀 셀 내부 줄바꿈 저장 방식을 수정했습니다.

## 핵심 수정
- 기존 CRLF(\r\n) 저장 방식 제거
- Excel Alt+Enter와 동일한 LF(CHAR(10)) 방식으로 저장
- [*3*] 추천번호는 직접 붙여넣기 테스트와 같은 `1)번호` 1줄 1조합 방식 유지
- 프론트에서 서버로 전달할 때도 CRLF 변환 없이 LF만 전달

## 수정 파일
- backend/app.py
- frontend/app.js

## 사용 방법
기존 프로젝트에 핵심 패치 파일을 같은 경로로 덮어쓴 뒤 서버를 재시작하세요.
