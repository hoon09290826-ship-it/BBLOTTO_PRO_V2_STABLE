# RC7-13 문자간다 공백 정렬 패치

- 문자간다에서 셀 내부 줄바꿈이 제거되어 추천번호가 붙는 문제 대응
- [*3*] 추천번호를 줄바꿈 의존 방식에서 공백 정렬 방식으로 변경
- 예: [추천번호]                            1.  3, 18, 21...              2. 10, 17...
- XLS 생성 시 [*3*] 셀은 줄바꿈을 넉넉한 공백으로 변환해 저장
- 기존 [*1*], [*2*], [*4*]는 기존 줄바꿈 처리 유지

적용 파일:
- frontend/app.js
- backend/app.py
