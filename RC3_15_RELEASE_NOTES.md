# BBLOTTO PRO V2 Stable RC3-15

## 목적
당첨번호 저장 회차 무결성 수정.

## 수정 내용
- 추천 회차와 당첨번호 저장 회차를 분리했습니다.
- 추첨 전/미래 회차에 당첨번호가 저장되는 문제를 차단했습니다.
- 1231회 당첨번호가 1232회로 잘못 저장되는 문제를 방지했습니다.
- 서버 시작 시 잘못 저장된 미래 회차 당첨번호를 자동 정리합니다.
- 수동 저장/API 저장/회원별 자동확인 모두 저장 가능 최신 완료 회차까지만 허용합니다.
- `/api/rc3-15/draw-integrity` 점검 API 추가.
- `/api/rc3-15/repair-draws` 복구 API 추가.
- `/api/version`을 RC3-15로 갱신했습니다.

## 기대 결과
- 최근 저장 당첨번호에는 실제 추첨 완료 회차만 표시됩니다.
- 1232회는 추천 생성 회차로만 사용되고, 추첨 전에는 당첨번호 DB에 저장되지 않습니다.
